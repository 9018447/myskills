# Clapeyron.jl - Fluid Thermodynamics Toolkit

An extensible, open-source fluid thermodynamics toolkit for Julia that provides a comprehensive library of thermodynamic models and a framework for implementing custom models.

## Quick Start

### Installation in Julia

```julia
# Enter Pkg mode with ]
] add Clapeyron

# Start using
using Clapeyron
```

### Installation in Python

```bash
pip install pyclapeyron
# or
uv add pyclapeyron
```

### Basic Usage Example

```julia
using Clapeyron

# Define a model (e.g., Peng-Robinson for propane)
model = PR(["propane"])

# Calculate properties
T = 300.0  # Temperature in K
P = 1e6    # Pressure in Pa

# Calculate saturation properties
T_sat, P_sat, vl, vv = saturation_pressure(model, T)

# Calculate enthalpy
h = enthalpy(model, P, T)

# Flash calculation
z = [0.5, 0.5]  # Composition
model_mixture = PR(["propane", "butane"])
result = tp_flash(model_mixture, P, T, z)
```

## Overview

Clapeyron.jl is a comprehensive thermodynamic modeling package that provides:

- **30+ Equation of State families** including cubic, SAFT, activity, and empirical Helmholtz models
- **Comprehensive parameter database** with 185+ CSV files for thousands of components
- **Phase equilibrium calculations** for VLE, LLE, VLLE, and electrolyte systems
- **Bulk property calculations** including enthalpy, entropy, heat capacity, and more
- **Flash algorithms** for P-T, P-H, T-S, and other property specifications
- **Extensible architecture** for custom models and methods
- **Automatic differentiation support** for efficient gradient calculations
- **Python interface** via pyclapeyron package

## Usage

### Model Construction

#### Single Component Models

```julia
using Clapeyron

# Peng-Robinson for propane
model = PR(["propane"])

# PC-SAFT for water
model = PCSAFT(["water"])

# NRTL activity model
model = NRTL(["acetone", "water"])
```

#### Mixture Models

```julia
# Binary mixture with PR
model = PR(["propane", "butane"])

# Ternary mixture with PC-SAFT
model = PCSAFT(["methane", "ethane", "propane"])

# Specify composition
z = [0.5, 0.3, 0.2]
```

#### Group Contribution Models

```julia
# UNIFAC (automatic group contribution)
model = UNIFAC(["acetone", "water"])

# SAFT-γ Mie
model = SAFTgammaMie(["propane", "butane"])
```

### Pure Component Calculations

```julia
using Clapeyron

model = PR(["propane"])
T = 300.0  # K

# Saturation properties
T_sat, P_sat, v_l, v_v = saturation_pressure(model, T)

# Properties at specified conditions
P = 1e6  # Pa
h = enthalpy(model, P, T)
s = entropy(model, P, T)
cp = heat_capacity(model, P, T)
Z = compressibility_factor(model, P, T)

# Critical point
T_c, P_c, V_c = crit_pure(model)
```

### Mixture Phase Equilibria

```julia
using Clapeyron

# Binary mixture
model = PR(["propane", "butane"])
z = [0.5, 0.5]  # Overall composition
T = 350.0  # K
P = 1e5   # Pa

# PT Flash
result = tp_flash(model, P, T, z)

# Bubble point
P_bubble, y = bubble_pressure(model, T, z)

# Dew point
P_dew, x = dew_pressure(model, T, z)
```

### Phase Diagrams

```julia
using Clapeyron
using Plots

model = PR(["ethanol", "water"])

# P-x-y diagram at constant T
P = 101325  # Pa
x = range(0, 1, length=100)
Pxy = Pxy_data(model, T, x)

plot(x, Pxy.P, label="Bubble")
plot!(x, Pxy.Pdew, label="Dew")

# T-x-y diagram at constant P
Txy = Txy_data(model, P, x)
plot(x, Txy.T, label="Bubble")
plot!(x, Txy.Tdew, label="Dew")
```

### Advanced: Custom Model Implementation

```julia
using Clapeyron

# Define a custom cubic EoS
struct MyCubic <: EoSModel
    components::Vector{String}
    params::MyParams
end

# Implement required methods
function Clapeyron.a_eos(model::MyCubic, V, T, z)
    # Return a parameter
    return value
end

function Clapeyron.pressure(model::MyCubic, V, T, z)
    # Return pressure
    return R * T / (V - b) - a / (V^2 + 2*b*V - b^2)
end

# Use the model
model = MyCubic(["propane"])
```

## Troubleshooting

### Common Issues

#### "Component not found in database"
```julia
# Solution 1: Check component name spelling
# Use Clapeyron.get_metadata() to see available components

# Solution 2: Provide user-defined parameters
user_locations = ["my_params.csv"]
model = PR(["custom_comp"]; user_locations)
```

#### "Flash calculation failed to converge"
```julia
# Try different initial guesses
result = tp_flash(model, P, T, z;
    flash = :Michelsen,  # or :PhaseSplit
    x0 = [0.5, 0.5]      # initial phase composition
)

# Relax tolerances
using Clapeyron: TPFlashOptions
options = TPFlashOptions(rtol = 1e-6)
```

#### "Poor results near critical point"
```julia
# Use more sophisticated methods
result = tp_flash(model, P, T, z;
    flash = :GlobalOpt    # Global optimization
)

# Check stability
stable = stability_test(model, P, T, z)
```

### Performance Tips

   ```julia
   using StaticArrays
   z = @SVector [0.5, 0.5]
   ```

   ```julia
   # First run is slow (compilation)
   # Subsequent runs are fast
   ```

   - Default van der Waals mixing is fast
