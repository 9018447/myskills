# Clean Data to CSV

将不规则数据格式转换为干净的标准CSV格式，专门为数据可视化工具（如vega-lite、D3.js）准备数据。

## 何时使用

- 用户需要清洗科学数据或实验数据
- 数据包含LaTeX符号、上标、下标等需要转换为Unicode
- 数据来自论文、实验软件导出或格式化文本
- 需要为数据可视化准备标准CSV格式
- 多组数据需要合并到一个表格中

## 核心任务

1. **提取标题和数据**：识别并提取title和data部分，移除无关符号
2. **符号转换**：将LaTeX数学符号、上标下标转换为Unicode
3. **格式统一**：所有数据放在同一行，使用逗号分隔
4. **处理空值**：保留空值（两个连续逗号 `,,`）
5. **输出CSV**：返回可直接保存为.csv文件的格式

## 常见符号转换

### 上标/下标
- `^E` → `ᴱ` (上标E)
- `^-` → `⁻` (上标减号)
- `^+` → `⁺` (上标加号)
- `^0` → `⁰`, `^1` → `¹`, `^2` → `²`, `^3` → `³`
- `_i` → `ᵢ`, `_j` → `ⱼ` (下标)

### LaTeX数学符号
- `\alpha` → `α`, `\beta` → `β`, `\gamma` → `γ`, `\delta` → `δ`
- `\sum` → `∑`, `\prod` → `∏`, `\int` → `∫`
- `\infty` → `∞`, `\pm` → `±`, `\mp` → `∓`
- `\leq` → `≤`, `\geq` → `≥`, `\neq` → `≠`
- `\approx` → `≈`, `\cdot` → `·`, `\times` → `×`
- `\frac{a}{b}` → `a/b`

### 其他格式符号
- `→` 或 `->` → 删除或保留（根据上下文）
- 多余的空格、制表符 → 单个空格
- 行号（如 `1→`）→ 删除

## 数据格式处理

### 多曲线数据
当遇到多组数据（如多个Curve）时：

**选项1：长格式（推荐用于vega-lite）**
```csv
curve,x,y
Gᴱ,0.0,0.0
Gᴱ,0.004,-0.04464358179776195
Gᴱ,0.02,-0.2003598184772693
Hᴱ,0.0,0.0
Hᴱ,0.004,-0.08718178526068515
-TSᴱ,0.0,0.0
-TSᴱ,0.004,0.0425382034629232
```

**选项2：宽格式（适合并排比较）**
```csv
x,Gᴱ,Hᴱ,-TSᴱ
0.0,0.0,0.0,0.0
0.004,-0.04464358179776195,-0.08718178526068515,0.0425382034629232
0.02,-0.2003598184772693,-0.4006340399212256,0.2002742214439563
```

根据用户的具体需求选择合适格式。如果没有明确指定，默认使用**长格式**（更适合vega-lite）。

### 缺失值处理
- 保留为空：两个连续逗号 `,,`
- 不要用 NULL、N/A 或 0 填充（除非用户明确要求）

## 输出规范

1. **第一行**：列名（header）
2. **数据行**：每行一个数据点
3. **分隔符**：逗号 `,`
4. **无额外内容**：不要添加Markdown代码块、解释文字或其他内容
5. **纯文本**：直接返回CSV内容，可被用户直接复制到文件

## 示例转换

### 输入
```
Curve G^E
0.0 0.0
0.004 -0.04464358179776195

Curve H^E
0.0 0.0
0.004 -0.08718178526068515
```

### 输出
```csv
curve,x,y
Gᴱ,0.0,0.0
Gᴱ,0.004,-0.04464358179776195
Hᴱ,0.0,0.0
Hᴱ,0.004,-0.08718178526068515
```

## 处理流程

1. **读取输入数据**：理解数据格式和结构
2. **识别标题**：提取曲线名称、列名等标题信息
3. **提取数据**：解析数值数据点
4. **转换符号**：应用符号转换规则
5. **组织数据**：根据数据结构选择合适的CSV格式
6. **输出结果**：返回纯CSV文本，无额外格式

## 注意事项

- 保持数值精度，不要四舍五入
- 处理科学计数法（如 `1.23e-5`）
- 识别并处理注释行（以 `#` 开头）
- 处理不同的分隔符（空格、制表符、逗号等）
- 如果数据格式不明确，询问用户期望的输出格式
