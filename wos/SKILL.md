# Web of Science Skills Bundle

Web of Science 技能复合包，整合 WoS 相关原子技能，提供完整的文献调研工作流支持。

## 子技能

- **wos-search** — Web of Science 文献搜索
- **wos-paper-detail** — 论文详情获取
- **wos-navigate-pages** — 搜索结果分页浏览
- **wos-parse-results** — 搜索结果解析
- **wos-download** — PDF 全文下载
- **wos-export** — 引用导出（Zotero/RIS/BibTeX/Excel）
- **wos-paper-workflow** — 完整论文工作流
- **wos-latex-citation-manager** — LaTeX 自动引用管理

## 使用场景

- Web of Science 文献检索与调研
- 批量获取论文元数据和 PDF
- 构建参考文献库并导出到文献管理工具
- 为 LaTeX 文档自动添加 WoS 参考文献
