# Gnuplot 科研绘图 Skill

Gnuplot 是一个强大且轻量的命令行绘图工具，自 1986 年以来持续开发，广泛用于科研数据可视化。

## 核心能力

- 2D/3D 科学图表绘制
- 函数曲线与数据拟合
- 热力图、等高线图、矢量图
- 直方图、误差条图
- 多子图布局与出版质量输出（PDF/EPS/PNG）

## 使用场景

- 绘制科学图表与数据可视化
- 生成 gnuplot 代码与脚本
- 将数据文件绘制成图表
- 创建出版质量的科研图片
- 数学函数可视化与曲线拟合
